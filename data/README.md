# Dataset utilizado

Para este proyecto se utilizó el dataset Iris incluido en la librería Scikit-Learn. No requiere archivo externo ya que se carga mediante la función load_iris().

## Descripción

El dataset contiene información sobre flores de tres especies:

- Iris Setosa
- Iris Versicolor
- Iris Virginica

Cuenta con 150 registros y 4 variables numéricas:

- Largo del sépalo
- Ancho del sépalo
- Largo del pétalo
- Ancho del pétalo

## Fuente
